# Instagram Profile Picture Resolution Limitation

## สถานการณ์ปัจจุบัน (March 2026)

Instagram ได้เปลี่ยนแปลง API และจำกัดความละเอียดของรูปโปรไฟล์ที่สามารถดาวน์โหลดได้เป็น **640x640 พิกเซล** เท่านั้น

### ประวัติการเปลี่ยนแปลง

- **ก่อนหน้านี้**: สามารถดาวน์โหลดรูปโปรไฟล์ได้ที่ความละเอียด 1080x1080 พิกเซล
- **ปัจจุบัน (March 2026)**: Instagram API ส่งคืนความละเอียดสูงสุดเพียง 640x640 พิกเซล

### วิธีที่เราพยายามแก้ไข

เราได้ลองหลายวิธีเพื่อหาทางเข้าถึงรูปความละเอียดสูง:

1. ✅ **Anonymous API** - ได้ 640px
2. ✅ **Authenticated API (with sessionid)** - ยังคงได้ 640px
3. ❌ **Third-party scrapers** - ส่วนใหญ่ล้มเหลวหรือถูกบล็อก
4. ❌ **yt-dlp** - Instagram profile support ถูกทำเครื่องหมายว่า "broken"
5. ❌ **URL manipulation** - การเปลี่ยน s640x640 เป็น s1080x1080 ไม่ทำงาน

### สรุป

นี่เป็นข้อจำกัดจาก Instagram เอง ไม่ใช่ปัญหาของโปรแกรมเรา เว็บไซต์ดาวน์โหลดรูปโปรไฟล์ Instagram อื่นๆ ก็ประสบปัญหาเดียวกัน

### แนวทางปัจจุบัน

โปรแกรมจะ:
- ใช้ Anonymous API (ไม่ต้องใช้ sessionid)
- แสดงความละเอียดที่แท้จริงในปุ่มดาวน์โหลด (640px)
- ดาวน์โหลดรูปโปรไฟล์ในความละเอียดสูงสุดที่ Instagram อนุญาต
- มี HTML scrape เป็น fallback

### หมายเหตุ

- **ไม่จำเป็นต้องใช้ IG_SESSIONID** เพราะ authenticated API ก็ให้ความละเอียดเท่ากับ anonymous API
- ถ้า Instagram เปลี่ยนนโยบายในอนาคตและอนุญาตให้ดาวน์โหลดความละเอียดสูงขึ้น โปรแกรมจะสามารถดาวน์โหลดได้ทันทีโดยไม่ต้องแก้ไขโค้ด
